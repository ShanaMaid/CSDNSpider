React的起源

React 起源于 Facebook 的内部项目，因为该公司对市场上所有 JavaScript MVC 框架，都不满意，就决定自己写一套，用来架设 Instagram 的网站。做出来以后，发现这套东西很好用，就在2013年5月开源了。由于 React 的设计思想极其独特，属于革命性创新，性能出众，代码逻辑却非常简单。所以，越来越多的人开始关注和使用，认为它可能是将来 Web 开发的主流工具。


React的优势

在学习的时候看了很多文档，把React说得非常复杂，用了很多专业性的词汇，实际上更适合有一定基础的人阅读。对于新手小白来说，只用知道以下几点就可以了

  • React解决了页面数据频繁刷新导致的DOM操作的性能瓶颈问题
  • React把一个页面进行了组件化拆分，每个组件都有独立的逻辑
  • React中的组件可以重复调用，一个组件可以使是他其他的组件的一部分，解决了代码冗余问题

总的来说，一个页面就是可以看成是由很多的组件组成的，它们共同组合形成了一个页面。

可能有人对ReactDOM.render这里的写法很疑惑，其实这样写是为了美观，实际上应该是这样的

  ReactDOM.render( <MessageBox />,document.getElementById('example') );

React组件间值传递的问题

在上面的例子中可以看到我们都是把内容直接写在了组件里面，很明显这不符合我们组件复用的初衷，所以我们需要进行修改。

这里用到的是Props属性，我们直接举例吧，对组件再次进行修改吧

      var LeftItem = React.createClass({
	  render: function() {
	    return (
	      <li>{this.props.content}</li>
	    );
	  }
	  });

	  var RightItem = React.createClass({
	  render: function() {
	    return (
	      <li>{this.props.content}</li>
	    );
	  }
	  });

      var LeftBox = React.createClass({
	  render: function() {
	    return (
	      <div>
	      	<ul>
	      		左边盒子
	      		<LeftItem content = "我是左边传入的内容1"/>
	      		<LeftItem content = "我是左边传入的内容1"/>
	      	</ul>
	      </div>
	    );
	  }
	  });


	  var RightBox = React.createClass({
	  render: function() {
	    return (
	      <div>
	      	<ul>
	      		右边盒子

	      		<RightItem  content = "我是右边传入的内容2"/>
	      	</ul>
	      </div>
	    );
	  }
	  });

React对于敏感字符的处理


传值的时候还会有个问题就是，如果我再content里面写的是敏感字符呢？比如标签会怎么样？


我们试试


 var LeftBox = React.createClass({
	  render: function() {
	    return (
	      <div>
	      	<ul>
	      		左边盒子
	      		<LeftItem dangerouslySetInnerHTML = "<p>This is <em>another</em> comment</p>" />
	      		<LeftItem content = "我是左边传入的内容1"/>
	      	</ul>
	      </div>
	    );
	  }
	  });







结果







我们发现并没有出现我们担心的情况，为什么呢？


React默认会进行HTML的转义，避免XSS攻击，如果要不转义，可以这样






var RightBox = React.createClass({
	  render: function() {
	    return (
	      <div>
	      	<ul>
	      		右边盒子
	      		<span dangerouslySetInnerHTML= {{__html: "<h1>我是右边传入的内容1<h1>" }} />
	      		<RightItem  content ="我是右边传入的内容1" />
	      		<RightItem  content = "我是右边传入的内容2"/>
	      	</ul>
	      </div>
	    );
	  }
	  });


即加入属性



dangerouslySetInnerHTML= {{__html: string }}
















React中组件的初始化函数getInitialState()




getInitialState()在组件的生命周期中仅执行一次，设置组件的初始化状态。




一般用来初始化组件中的值，设置默认值


eg:




var Test = React.createClass({
	  getInitialState: function() {
	    return {data: "初始值"};
	  },
	  render: function() {
	    return (
	      <div>
	      	{this.state.data}
	      </div>
	    );
	  }
	  });

	  ReactDOM.render(
	    <Test />,
	    document.getElementById('example')
	  );
