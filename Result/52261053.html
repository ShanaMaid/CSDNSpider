首先我们新建一个文件夹，这里取名为test吧，然后再文件夹里新建一个less文件，取名也叫test吧

然后我们把这个文件夹拖到koala的右侧方框中得到如下



在less文件中写入

@base: #f938ab;  
  
.box-shadow(@style, @c) when (iscolor(@c)) {  
  -webkit-box-shadow: @style @c;  
  box-shadow:         @style @c;  
}  
.box-shadow(@style, @alpha: 50%) when (isnumber(@alpha)) {  
  .box-shadow(@style, rgba(0, 0, 0, @alpha));  
}  
.box {  
  color: saturate(@base, 5%);  
  border-color: lighten(@base, 30%);  
  div { .box-shadow(0 0 5px, 30%) }  
}


在koala选中test文件，右侧会弹出侧边栏，选中自动编译，点击执行编译，会自动生成一个同名的css文件，当less文件更新后，koala又打开的时候，css文件会自动更新！注意，需要保持koala是打开状态。同时如果less出现语法错误，右下角的koala图标会报错
