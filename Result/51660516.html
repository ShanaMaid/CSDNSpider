今天复习数据库的时候忘记了identity的用法，查了一下百度，发现很多人把这个问题说得很复杂。

翻了翻书又实践了下，发现其实很简单。

比如

create table tb(id int identity(1,1),
			name varchar(12)
)
这个SQL语句的意思是创建一个叫tb的表，表中有个id的int字段，关键来了，identity(1,1),
其实就是代表id字段从1开始自增，每次增加1
如果是identity(2,2)就是代表从2开始，每次自增2
同时插入数据的时候不必给id赋值了，它会自动赋值。
比如insert tb values('','Jack');
	
insert tb values('','Tom');
得到的结果
id 	name
1	Jack
2	Tom
即标识列， identity(a,b)，ab均为正整数，a表示开始数，b表示增幅

ab不填的时候默认从1开始
